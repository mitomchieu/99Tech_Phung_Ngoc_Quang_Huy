# Todo App Backend

This is a simple Todo app backend built with TypeScript, Express.js, and SQLite.

## Features

- Create, read, update, and delete todos.
- List todos with basic filtering (e.g., by completion status).
- Uses an in-memory SQLite database.

## Installation

1. Clone the repositor
2. Install dependencies:
   `npm install`
3. Run the development server:
   `npm run dev `
